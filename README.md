# 🍕 Pizza Mia! Pizzeria

Check out the live demo of my project here: [Live Site](https://pizza.aldiresee.com/)


✨ Features
🍽️ Menu & Ordering
Interactive Menu - Browse pizzas, salads, breakfast items, and desserts

Product Customization - Choose toppings, crust types, coffee options with real-time price updates

Visual Feedback - Images change based on selected options

Shopping Cart - Add/remove items, adjust quantities, see real-time totals

📅 Table Booking
Interactive Floor Plan - Click on available tables to select

Date & Time Picker - Choose your preferred dining time

Duration Selection - Book tables for 1-4 hours

Starter Options - Add water or bread to your reservation

Real-time Availability - See booked tables instantly update

👤 User Experience
Responsive Design - Works perfectly on mobile, tablet, and desktop

Real-time Updates - Cart and booking availability update instantly

Visual Feedback - Smooth animations for all interactions

Customer Reviews - Testimonials carousel

🛠️ Technology Stack
Frontend
HTML5 - Semantic markup

SCSS - Custom styles with variables and mixins

Vanilla JavaScript (ES6+) - No frameworks, pure JS

Handlebars.js - Template rendering

Flickity - Touch-responsive carousel

Flatpickr - Date picker

rangeSlider - Time selection slider

Backend
Supabase - PostgreSQL database, authentication, and real-time capabilities

REST API - Custom endpoints for orders and bookings
